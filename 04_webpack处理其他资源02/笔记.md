在webpack5之前加载资源使用raw-loader,url-loader,file-loader
在webpack5开始，我们可以使用资源模块类型（asset module type）来替代上面的这些loader

asset/resource   发送一个单独url  file-loader
asset/inline   url-loader
asset    可以直接设置maxSize
 test: /\.(png|jpg|gif|jpeg)$/,
                // type: "asset/resource",
                type: 'asset',
                generator: {
                    filename: 'img/[name][hash:8][ext]'
                },
                parser: {
                    dataUrlCondition: {
                        maxSize: 10 * 1024 // 小于10kb的图片转化成base64
                    }
                }
